# Sanggala Corridor Project - Dashboard

Situs monitoring untuk Sanggala Corridor Project, berisi kumpulan dashboard pemantauan lapangan.

🔗 **Live site:** https://gis-carbonx.github.io/dashboard/

## Struktur Halaman

| File | Keterangan |
|---|---|
| `index.html` | Halaman utama (landing page) |
| `list.html` | Daftar semua dashboard monitoring |
| `patrol.html` | Peta & data patroli lapangan |
| `bio-monitoring.html` | Monitoring biodiversity (camera trap+ bioacoustic) |
| `planting.html` | Monitoring kegiatan penanaman/agroforestri |
| `fire-monitoring.html` | Fire monitoring (embed Looker Studio) |
| `land-clearing-alert.html` | Land clearing alert / GFW (embed Looker Studio) |
| `community-fire-management.html` | Community-based fire management (embed Looker Studio) |
| `satellite-tracker.html` | Satellite hotspot (NASA FIRMS) tracker |
